 Airline Operations Data Science Project
Overview

End-to-end data science project analyzing airline flight operations to study pricing, flight duration, and delay patterns.

Dataset

10,000+ airline flight records

Includes airlines, airports, timings, ticket prices, and flight status

Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

Key Steps

Data cleaning & feature engineering

Exploratory Data Analysis (EDA)

Machine learning (Logistic Regression, Random Forest)

Model evaluation & insights

Key Insights

Ticket price varies by airline and flight duration

Departure time impacts delays

Random Forest outperformed Logistic Regression
